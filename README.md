# BeTube - Nostalgia Yourself

A fully functional video-sharing platform.

## Live Demo
Visit the website here: **[betube.duckdns.org](http://betube.duckdns.org)**

---

## Features
* **Retro Media Player:** Watch videos using a classic mid-2000s styled player.
* **User Channels:** Create your own channel and customize your presence.
* **Video Uploads:** Upload local MP4 files directly to the platform.
* **Community Interactions:** Leave comments, view counts, and rate videos using the vintage 5-star system.
* **Site Rules:** Built-in community guidelines popup for all visitors.
